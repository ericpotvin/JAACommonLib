# Java Android Application Common Libraries

## Requirements

- Gradle 2.2.3
- JDK 1.7

## Build

Clean

    Gradle projects -> :commonlib -> Tasks -> build -> clean

To build/compile

    Gradle projects -> :commonlib -> Tasks -> build -> assembleRelease

Output file:

    ./commonlib/build/outputs/aar/JAACommonLib-release.aar
